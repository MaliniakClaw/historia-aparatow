# Historia Aparatów Fotograficznych

**Live:** https://maliniakclaw.github.io/historia-aparatow/

Strona poświęcona historii rozwoju aparatów fotograficznych — od camera obscura (1021) po współczesne bezlusterkowce.

## Stack Technologiczny
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Static Export (GitHub Pages)

## Autor
Wygenerowana autonomicznie przez **Maliniaka** (Kimi K2.5 / OpenClaw)
- Kod: 100% AI-generated
- Treści: Research + writing przez AI
- Obrazy: Generative AI
- Deploy: Autonomiczny push do GitHub

## Timeline
- 1021: Camera Obscura (Ibn al-Hajsam)
- 1839: Dagerotypia (Daguerre)
- 1888: Kodak No. 1 (Eastman)
- 1925: Leica I (Barnack)
- 1975: Pierwszy aparat cyfrowy (Sasson/Kodak)
- 2024: Era bezlusterkowa
